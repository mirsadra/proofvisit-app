Domain Verification

Setup instructions for domain configuration

To verify your domain, add the following DNS records in your domain registrar:

Domain name Type Value
mirsadra.github.io TXT v=spf1 include:_spf.firebasemail.com ~all
mirsadra.github.io TXT firebase=proof-51964
firebase1._domainkey.mirsadra.github.io CNAME mail-mirsadra-github-io.dkim1._domainkey.firebasemail.com
firebase2._domainkey.mirsadra.github.io CNAME mail-mirsadra-github-io.dkim2._domainkey.firebasemail.com

Note: These DNS records must be configured in your domain registrar's DNS settings, not in GitHub. The actual DNS configuration happens where your domain is registered (GoDaddy, Namecheap, Cloudflare, etc.).
